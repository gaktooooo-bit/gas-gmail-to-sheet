# ラベル設定・Gmailフィルタ・抽出ルールの作り方

実データなしでデモ・動作確認を再現するための構成例です。

## 1. Gmailラベルを作る

Gmail左メニュー下部の「ラベルを作成」から、以下2つを作成します。
- `自動転記`（処理対象）
- `転記済み`（処理が終わると自動で付く。手動作成は任意＝なければスクリプトが作成）

## 2. Gmailフィルタで自動ラベリング（任意・推奨）

`設定 → フィルタとブロック中のアドレス → 新しいフィルタを作成`
- 例：差出人 `order@example-shop.com`、または件名に `ご注文ありがとうございます`
- アクション：**ラベル「自動転記」を付ける**

これで対象メールが届くたびに自動で対象化され、完全自動で回ります。

## 3. 抽出ルール（PARSE_RULES）の考え方

メール本文の決まった文言を正規表現で拾います。`( )` の中身が抽出値です。

| 取り出したいもの | 本文の例 | 正規表現の例 |
|------------------|----------|--------------|
| 注文番号 | `注文番号: A-12345` | `注文番号[\s:：]*([A-Za-z0-9\-]+)` |
| 金額 | `合計金額：12,800円` | `(?:金額\|合計)[\s:：]*([0-9,]+)\s*円` |
| 電話番号 | `TEL: 03-1234-5678` | `(0\d{1,4}-\d{1,4}-\d{3,4})` |
| メール | `連絡先 taro@example.com` | `([\w.\-]+@[\w.\-]+\.\w+)` |

`Config.gs` の `PARSE_RULES` に `{ header: '見出し', pattern: '正規表現' }` で追加するだけ。
見出しがそのままシートの列名になります。

## 4. デモ用のテストメール

自分宛に以下のような本文のメールを送り、`自動転記` ラベルを付けます。

```
件名: 【テスト】ご注文ありがとうございます

山田様

ご注文を承りました。
注文番号: A-12345
合計金額：12,800円
お電話番号: 03-1234-5678

ありがとうございました。
```

## 5. デモのやり方
1. ラベルを作り、上のテストメールに `自動転記` を付ける。
2. スクリプトを設置し `Config.gs` を確認。
3. `processInbox` を手動実行（または `setupTrigger` で定期化）。
4. `受信ログ` シートに行が追加され、注文番号・金額・電話番号が抽出され、
   メールのラベルが `転記済み` に変わればOK。
